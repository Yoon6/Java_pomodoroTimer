# README

# Pomodoro Timer

자바 스윙을 이용한 포모도로 타이머. Windows프로그램이다.

이 프로그램은 윈도우 스토어의 'Pomy'를 보고 만듬.

---

# 프로젝트 정보

- 시작, 설정, 닫기를 할 수 있다.

- 시간 설정을 할 수 있다.
